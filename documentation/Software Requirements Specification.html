Software Requirements Specification

1.                  Introduction

1.1               Purpose

This SRS describes the specifications for the social trend board – strebo. The document will provide an overview over the „Use Case Diagram“ and explains the used technologies.

1.2               Scope

Strebo is a web-based application that combines the trends of various social media networks on one single page. Furthermode the user will be able to connect it's own social media accounts to get access to a personal dashboard, to view it's personally relevant content.

1.3               Definitions, Acronyms and Abbreviations

strebo – social trend board

1.4               References

Wordpress

GitHub

Heroku-Web-App

1.5               Overview

First, this document will provide an overall description. Afther that the requirements will be specified and at the end there will be supporting information.
